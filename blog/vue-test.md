# NightWatch

NightWatch是一个专门的端对端测试运行器（runner）。

它依赖于浏览器控制器selenium，而selenium是一个`.jar`后缀的文件，需要java的运行环境。所以你需要安装java并配置好环境变量。然而，selenium需要对应的driver配合来操控浏览器。
